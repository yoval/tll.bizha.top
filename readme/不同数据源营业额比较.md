### 不同数据源营业额比较

- 不同数据源因计算方式不同，营业额对比会有差异。


![](https://images.bizha.top/202511131344192.png)


- 这些数据均是通过收银机的数据，还有部分数据未通过收银机，如部分团购数据。这些数据通过手动补录到`imp_online_new_channel_supplement`。


- 不同计算逻辑也可能会导致营业额差异，例如：

`ads_dbs_trade_food_di` 的`sum(dp_total_amount)` 在10月的营业营业额为239025657.93元，而通过`sum(销量)*sum(标准价)`计算的营业额为222011377.92元, 差异为17014280.01元, 差异率为7.12%。
